</li><li>Safe Harbor
</li><li>Jericho Forum Commandments
</li><li>Heartbleed patched

</p><ul><li>Founded
</li><li>Holding
</li><li>Domain Registration
</li><li>FedRAMP level
</li><li>FISMA</li></ul><p>Due to the changes in this message:
